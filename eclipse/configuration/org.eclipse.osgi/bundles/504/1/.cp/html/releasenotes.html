IAR Embedded Workbench for Eclipse (ARM 6.50.x/6.60.x/6.70.x) Release Notes

If you are upgrading from 0.13.x or earlier versions of the plugin you must UNINSTALL all IAR plugins before upgrade to 6.50.x.

The plugin supports IAR Embedded Workbench for ARM (EWARM) 6.50.x/6.60.x/6.70.x. The plugin version major/minor number ("6.50") reflects the oldest EWARM version with which the plugin is compatible.

See the list of known and fixed issues.

Documentation

Debugger documentation, build option reference, project import guide, and tutorials are available as Eclipse help documents.

Limitations

  • Windows only.
  • The supported drivers are
    • J-Link/J-Trace
    • I-jet/JTAGjet
    • Simulator
  • Profiling is not supported.
  • RTOS plug-ins are not supported.
  • No context sensitive help.

Support and feedback

For support and feedback on IAR Embedded Workbench for Eclipse, send an email to support@iar.se.
